# kenkou-react-native-wrapper

Kenkou Measurement SDK for React Native

## Installation

```sh
npm install kenkou-react-native-wrapper
```

## Usage

```js
import { multiply } from "kenkou-react-native-wrapper";

// ...

const result = await multiply(3, 7);
```
